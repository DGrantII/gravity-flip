# Gravity Flip

A fun and challenging gravity-flipping game built with Phaser.js!

## Description

Gravity Flip is an arcade-style game where you control a square that can flip gravity to avoid incoming obstacles. The goal is to survive as long as possible without letting the square touch the left side of the screen. As you progress through levels, the obstacles become faster and more complex.

## Features

- Multiple levels with increasing difficulty
- Smooth gravity flipping mechanics
- Engaging soundtrack
- Simple, intuitive controls
- Retro pixel art style

## How to Play

1. Use the space bar or mouse click to flip gravity
2. Avoid the blocks coming from the right
3. Don't let the square touch the left side of the screen
4. Survive for the level timer to advance

## Controls

- **Space Bar** or **Mouse Click**: Flip gravity

## Installation & Running

1. Clone or download this repository
2. Open `index.html` in a modern web browser
3. The game will load automatically

## Requirements

- Modern web browser with JavaScript enabled
- No additional dependencies required (Phaser.js is loaded from CDN)

## Game Structure

- `index.html` - Main HTML file
- `game.js` - Game configuration and initialization
- `level.js` - Level definitions and block generation
- `menu.js` - Main menu scene
- `start_screen.js` - Start screen scene
- `instructions.js` - Instructions scene
- `assets/` - Game assets (images, sounds, etc.)

## Technologies Used
